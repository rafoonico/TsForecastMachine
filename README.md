# Forecast Shares

Projeto base para comparar modelos de previsão de participação (shares) em séries temporais.

## Estrutura

- `config/` – arquivos de configuração.
- `data/` – dados brutos, intermediários e processados.
- `src/` – código fonte com preparação de dados, modelos e avaliação.
- `notebooks/` – notebooks exploratórios.

## Uso

```bash
make train
```
